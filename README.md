# Concinnity SCORM Portfolio Repository

## Purpose of This Repository

This GitHub repository contains **sample SCORM packages** developed by **Concinnity** and is provided **solely for demonstration and portfolio purposes**.

The files in this repository are intended to showcase:
- Our instructional design capability
- Our approach to learner-centred, accessible, and engaging eLearning
- The structure, interactivity, and technical quality of our SCORM builds
- Examples of microlearning, onboarding, compliance, and skills-based modules

This repository is **not** intended to be used as a learning library, training resource, or content distribution platform.
